About Shade

Shade is a web proxy, that allows users to evade filters and censorship and access blocked websites.

It's designed to be extremely fast and compatible: It processes the data on-the-fly, and it only modifies URLs when necessary - relative ones will "just work".

Shade is written in JavaScript with node.js. The source code is available on github.
